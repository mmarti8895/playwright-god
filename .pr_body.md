## Summary
- add feature-aware repository analysis and reusable memory maps to the CLI index workflow
- switch offline generation and auth/logging helpers from TypeScript assumptions to Python Playwright output
- add deterministic unit and integration coverage for feature inference, memory-map reuse, self-analysis, auth, and logging flows

## Why
The tool needed to remember repository structure at a feature level, reuse that memory efficiently, and emit Playwright tests in Python to match the product direction and test harness.

## Impact
- `playwright-god index` now prints inferred feature areas and stores feature metadata in saved memory maps
- `playwright-god generate` now produces Python Playwright output and reuses saved feature-aware memory
- `playwright-god plan` now works with the richer memory-map format and aligned Python-oriented guidance
