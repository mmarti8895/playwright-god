# Desktop screenshots

Drop PNG screenshots here using the filenames referenced from the root
`README.md` (e.g. `01-home.png`, `02-run.png`, …). Capture them at 2× DPI
on macOS and downscale, or use `gnome-screenshot -w` on Linux.

Keep each file under ~500 KB; run `pngquant --quality=70-90` if needed.
